# 多模型支持提示词

## 模型选择与适配

### 模型特点
1. **通义千问 (qwen-turbo)**：
   - 擅长中文理解和生成
   - 支持长文本处理
   - 适合知识梳理和总结

2. **通义千问增强版 (qwen-plus)**：
   - 更强大的推理能力
   - 适合复杂问题解决
   - 支持数学推导和逻辑分析

3. **通义千问代码版 (qwen-coder)**：
   - 擅长编程相关问题
   - 支持代码生成和解释
   - 适合技术学习场景

### 模型选择策略
1. 根据任务类型选择模型：
   - 知识学习/复习：qwen-turbo
   - 复杂问题解决：qwen-plus
   - 编程学习：qwen-coder

2. 根据输入复杂度选择模型：
   - 短文本、简单问题：qwen-turbo
   - 长文本、复杂推理：qwen-plus

3. 根据输出要求选择模型：
   - 普通文本：qwen-turbo
   - 代码、公式：qwen-coder

## 响应格式统一

### 输出结构要求
1. **学习内容**：
   - 清晰的层次结构
   - 重点知识点突出
   - 适当的例子说明

2. **复习计划**：
   - 明确的时间安排
   - 具体的复习内容
   - 可操作的复习方法

3. **习题生成**：
   - 题目+答案+解析的标准格式
   - 难度梯度合理
   - 覆盖关键知识点

## 任务适配规则

### 1. 学习任务
- **输入**：学习内容、用户当前水平
- **输出**：简化解释、知识框架、学习建议
- **模型**：优先使用qwen-turbo，复杂内容使用qwen-plus

### 2. 复习任务
- **输入**：学习历史、遗忘曲线数据
- **输出**：个性化复习计划、重点回顾内容
- **模型**：qwen-turbo

### 3. 习题生成
- **输入**：知识点、难度、数量
- **输出**：多样化习题、详细解析
- **模型**：qwen-turbo，编程题使用qwen-coder

### 4. 问题解答
- **输入**：具体问题
- **输出**：清晰、准确的解答
- **模型**：根据问题复杂度选择

## 提示词格式

### 通用格式
```
作为专业的学习助手，请根据以下要求完成任务：

任务类型：[学习/复习/习题生成/问题解答]
用户水平：[初级/中级/高级]
知识点：[具体知识点]

请按照以下结构输出：
1. [第一部分]
2. [第二部分]
...
```

### 模型特定格式

#### 通义千问代码版
```
作为专业的编程导师，请为以下知识点生成学习内容：

知识点：[具体编程知识点]
编程语言：[具体语言]
用户水平：[初级/中级/高级]

请包含：
1. 核心概念解释
2. 代码示例
3. 常见错误
4. 练习题目
```

#### 通义千问增强版
```
作为专业的知识导师，请深入解析以下复杂概念：

概念：[具体复杂概念]
学科：[具体学科]
用户水平：[初级/中级/高级]

请包含：
1. 概念的起源和发展
2. 核心原理
3. 与其他概念的关联
4. 实际应用案例
5. 批判性分析
```
