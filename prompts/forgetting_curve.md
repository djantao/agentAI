# 遗忘曲线复习计划提示词

## 遗忘曲线原理
- 学习后20分钟：遗忘42%
- 1天后：遗忘67%
- 1周后：遗忘75%
- 1个月后：遗忘79%
- 复习能延长记忆保持时间，需要在关键遗忘点进行复习

## 间隔重复复习计划生成

### 输入要求
1. **学习历史**：所有相关的学习和复习记录
2. **时间线**：每次学习的具体日期和时长
3. **知识点**：涉及的具体概念、公式、定理等
4. **掌握程度**：用户自我评估或系统记录的掌握情况

### 生成内容要求
1. **个性化复习计划**：基于遗忘曲线和学习历史
   - 近期需要复习的知识点（1-3天内）
   - 中期复习计划（4-14天）
   - 长期复习计划（15-30天）

2. **知识点优先级排序**：
   - 高优先级：容易混淆、掌握不牢固、重要性高的知识点
   - 中优先级：基本掌握但需要巩固的知识点
   - 低优先级：已熟练掌握的知识点

3. **复习方法建议**：
   - 针对不同类型知识点的复习策略
   - 每次复习的时间分配建议
   - 检测掌握程度的方法

4. **强化记忆的习题**：
   - 针对薄弱知识点的专项练习
   - 难度递增的习题序列
   - 答案和详细解析

## 复习效果评估
- 每次复习后记录掌握程度变化
- 根据复习效果调整后续复习间隔
- 识别长期记忆稳定的知识点，减少复习频率

## 个性化调整参数
- **记忆强度系数**：根据用户学习表现调整
- **复习效率系数**：考虑用户复习时间和效果
- **知识点重要性权重**：根据学科要求和考试重点调整
- **时间灵活性**：允许用户根据实际情况调整复习计划
